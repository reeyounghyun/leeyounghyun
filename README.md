# [사용 용도]
## App.js
- 애플리케이션의 주요 컴포넌트를 정의하는 파일
- 애플리케이션의 전체적인 레이아웃과 구조를 정의
- 다양한 하위 컴포넌트을 포함하고, 필요한 props를 전달 함
- 상태와 이벤트를 핸들링 추가할수 있음

## index.js
- 에플리케이션을 브라우저에 랜더링하는 파일
- ReactDOM.render()를 사용하여 React 컴포넌트를 HTML DOM에 랜더링 함
- 전역 스타일 또는 CSS 파일을 포함
